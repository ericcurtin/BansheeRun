# BansheeRun

A cross-platform Virtual Pacer / Banshee Runner app built with Flutter and Rust.

Race against your previous performances or AI-generated pace targets with real-time GPS tracking and OpenStreetMap visualization.

## Features

- **GPS Run Tracking**: Record your runs with accurate GPS tracking
- **Banshee Racing**: Race against your previous recorded runs
- **AI Pacer**: Set a target pace and race against a virtual pacer
- **Real-time Feedback**: Audio tones when you fall behind or pull ahead
- **OpenStreetMap**: Beautiful maps with offline tile caching support
- **Run History**: View all your past runs with detailed statistics
